Features
A concept that could allow learners to manage their own data

Description

Exploring a concept allowing learners to directly manage certain amount of the data held by their learnign provider.

In theory, this would prevent a large amount of work and rework that learning provider administrative teams have to complete each return period. It would also allow learning providers to capture more accurate and up-to-date data from learners.
